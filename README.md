# Hafiz Idris — Portfolio

Personal portfolio of Hafiz Idris, Full Stack Developer based in Johor, Malaysia.

🌐 **Live**: [portfolio-hafiz-idris.vercel.app](https://portfolio-hafiz-idris.vercel.app)

## Tech stack

- **Next.js** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** for animations
- **React Icons**

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
